# liushiyong.github.io

使用 OKR (Objectives and Key Results)，你可以参与战略和策略的制定分解，主动寻找实现路径，和团队成员互相对齐，把个体结合到一起，形成组织快速前进。每周一次的进展对齐，能让你更好地朝达成 OKR 的方向前进 💪
